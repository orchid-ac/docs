## Welcome

orchid01 is a finance-native LLM for hedge funds and institutional investors. The gateway is **OpenAI-compatible** (`/v1/chat/completions`, `/v1/models`, …) and **Anthropic-compatible** (`POST /v1/messages`) so you can use either SDK family with the same Orchid key and **`/v1`** base URL.

## Base URL

```
https://llm.orchid.ac/v1
```

Local development often uses `http://127.0.0.1:8000/v1`. Get your API key at [app.orchid.ac](https://app.orchid.ac). Connection details: [Authentication](/getting-started/authentication).
