## List models

```
GET https://llm.orchid.ac/v1/models
```

Returns available models for your API key. Prefer the catalog id **`orchid01`** in clients; some deployments also accept aliases such as **`orchid`** — confirm with your environment if you rely on aliases.

```bash
curl https://llm.orchid.ac/v1/models \
  -H "Authorization: Bearer orchid-your-key-here"
```

```json
{
  "object": "list",
  "data": [
    {
      "id":       "orchid01",
      "object":   "model",
      "owned_by": "orchid"
    }
  ]
}
```
