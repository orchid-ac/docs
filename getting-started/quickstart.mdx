## Setup

Get your API key from the [Orchid dashboard](https://app.orchid.ac) under **Settings → API Keys**.

<Info>
  The API is **OpenAI-compatible** (`/v1/chat/completions`, `/v1/models`, …) and **Anthropic-compatible** (`POST /v1/messages`). Use your Orchid key and a base URL that includes **`/v1`**. See [Authentication](/getting-started/authentication) for headers (`Bearer`, `x-api-key`, `anthropic-version`) and local vs production hosts.
</Info>

## Your first call

<CodeGroup>

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://llm.orchid.ac/v1",
    api_key="orchid-your-key-here",
)

r = client.chat.completions.create(
    model="orchid01",
    messages=[{"role": "user", "content": "Summarise this 10-K filing..."}],
)
print(r.choices[0].message.content)
```

```typescript TypeScript
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "https://llm.orchid.ac/v1",
  apiKey: process.env.ORCHID_API_KEY!,
});

const r = await client.chat.completions.create({
  model: "orchid01",
  messages: [{ role: "user", content: "Summarise this 10-K filing..." }],
});
console.log(r.choices[0]?.message?.content);
```

```bash curl
curl -s https://llm.orchid.ac/v1/chat/completions \
  -H "Authorization: Bearer orchid-your-key-here" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "orchid01",
    "messages": [{"role": "user", "content": "Summarise this 10-K filing..."}]
  }'
```

</CodeGroup>

## Anthropic Messages (Python)

Use the official Anthropic client with the same **`/v1`** base URL and **`model: "orchid01"`**. Full guide: [Anthropic SDK](/integrations/anthropic).

```bash
pip install anthropic
```

```python
import anthropic

client = anthropic.Anthropic(
    base_url="https://llm.orchid.ac/v1",
    api_key="orchid-your-key-here",
)

msg = client.messages.create(
    model="orchid01",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello"}],
)
print(msg.content[0].text)
```

Tool-use and images on **`/v1/messages`** are not supported yet — use **chat completions** for tools ([Tool calling](/guides/tool-calling)).

## Streaming

<CodeGroup>

```python Python
stream = client.chat.completions.create(
    model="orchid01",
    messages=[{"role": "user", "content": "Analyse this credit agreement..."}],
    stream=True,
)
for chunk in stream:
    piece = chunk.choices[0].delta.content or ""
    if piece:
        print(piece, end="", flush=True)
```

```typescript TypeScript
const stream = await client.chat.completions.create({
  model: "orchid01",
  messages: [{ role: "user", content: "Analyse this credit agreement..." }],
  stream: true,
});
for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0]?.delta?.content || "");
}
```

</CodeGroup>
